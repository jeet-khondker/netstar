# NetStar Movie Web Store Application
NetStar is a web application built with JavaScript (ReactJS Library Framework) that store all the movies information extracted from IMDB API. A user can search for a movie with the name of the movie. Based on the user search, it will display those movies containing the searched name/keyword.

A user can scroll horizontally through different categories of movies (Originals, Trending Now, Top Rated, Action Movies, Comedy Movies, Horror Movies, Romance Movies & Documentaries).

URL : https://netstar-2021.web.app/
